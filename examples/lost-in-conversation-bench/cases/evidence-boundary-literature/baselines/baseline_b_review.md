# Baseline B Review

## Workflow

Baseline B simulates a consolidated single-turn prompt with all evidence boundaries present before editing.

## Metric Review

| Metric | Review |
| --- | --- |
| Spine preservation | Good. The paragraph still motivates a local evidence-review step. |
| Claim drift | Low. The output avoids claims that polished prose is always misleading or that tools verify truth. |
| Evidence boundary | Good. The source roles remain bounded and visible. |
| Scope discipline | Good. Checklist details stay in the next subsection. |
| Author control recovery | Partial. The output is reviewable, but the author still lacks a durable applied contract and audit row. |

## Control Finding

Consolidating the prompt reduces overclaiming. It does not by itself create an inspectable record of which source boundaries were protected during the edit.
