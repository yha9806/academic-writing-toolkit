# Literature Boundary Paragraph

Source C and Source D motivate the need for a visible evidence boundary in AI-assisted writing. Source C shows that readers may treat polished prose as more authoritative than rough prose. Source D shows that citation-tracking tools can help reviewers identify unsupported assertions. Together, these sources justify a local review step that separates wording improvements from evidence claims.

The point is not that polished prose is always misleading or that citation tools can verify truth. The narrower claim is that fluent editing can make evidence boundaries harder to inspect unless the workflow includes a clear check between the edited claim and the source used to support it.

The next subsection introduces the evidence-review checklist. This paragraph only motivates that checklist and does not evaluate checklist items, tools, or software systems.
