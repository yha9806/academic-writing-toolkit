# /style — British English Check

Run `/style` to scan chapter drafts and reading notes for common US spellings when the project is configured for British English.

The skill uses `scripts/audit-british-english.py` and can apply conservative whole-word replacements such as `color` to `colour`, `analyze` to `analyse`, and `behavior` to `behaviour`.

Safe fixes are mechanical only. The skill does not rewrite argument, tone, citations, or paragraph structure.
