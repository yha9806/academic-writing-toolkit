# /read — Page-by-Page PDF Reading

Read academic PDFs in bounded batches and produce structured output: key points, exact quotes with page numbers, terms, summary, and brief connections to the project.

Use `/read literature/Example_Source.pdf p.1-5`, `next page`, or `read p.12`. The skill never records notes or searches the web unless the user explicitly asks.

Limits: 15 pages per invocation and 90 pages per conversation.
