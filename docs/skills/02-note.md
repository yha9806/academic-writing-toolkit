# /note — Structured Reading Notes

Record reading notes to `literature/reading_notes/{Author}_{ShortTitle}_NOTES.md`.

Each notes file follows `literature/reading_notes/_template_NOTES.md` and includes a single-line `**Source**:` record, status, relevance, detailed notes, key terms, and thesis connections.

The skill is append-only. It creates one file per source, updates `Last updated`, and never overwrites existing notes.

Example command:

```text
note: Smith's argument supports the methods section because it defines the evaluation criteria.
```
