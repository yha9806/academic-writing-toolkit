# README visual assets

These SVGs are the repository copies of the Academic Writing Toolkit README visual system.

- `awt-readme-hero.svg` — product promise and controlled-writing loop
- `awt-control-loop.svg` — source-to-release workflow with human gates
- `awt-product-surfaces.svg` — boundary between local skills, the Codex plugin, and the ChatGPT App

Editable Figma source: <https://www.figma.com/design/HhaFm0uorv5oS7MsezWDN5>

The SVG files are intentionally self-contained: no external fonts, images, scripts, or tracking resources are required to render them on GitHub.
